# ❌⭕ Tic Tac Toe Game using Python (Tkinter)

This project is a **GUI-based Tic Tac Toe game** built using Python and Tkinter.  
Two players can play the game by taking turns on the same system.

---

##  Features
- Interactive GUI using Tkinter
- Two-player mode (X vs O)
- Automatic win detection
- Winning cells highlighted
- Popup message for winner

---

##  Technologies Used
- Python
- Tkinter
